# Nuxt Feed app

appname Feed Slash

A simple Nuxt.js application that displays a feed of articles fetched from an external API.

## Features

- Fetches articles from a public API
- Displays articles in a clean, responsive layout

## Framework

- Nuxt 3

## Authentication

- Clerk

## Runtime

Bun.sh
(must not use npm and yarn)

## Libraries

- Axios
- Tailwind CSS

## Deployment

- Cloudflare Workers
